# Personal Budgeting Tracker

Aplikasi mobile untuk mencatat dan mengelola keuangan pribadi, dibangun dengan Flutter & Dart.

## Fitur

- Login dengan session management (SharedPreferences)
- Dashboard saldo total, pemasukan, dan pengeluaran
- Tambah transaksi (pemasukan/pengeluaran)
- Edit & hapus transaksi
- Filter transaksi: Semua / Pemasukan / Pengeluaran
- Kategori transaksi
- Data tersimpan secara lokal

## Tech Stack

- Flutter & Dart
- SharedPreferences
- intl (format mata uang Rupiah)
- uuid (generate ID unik)
